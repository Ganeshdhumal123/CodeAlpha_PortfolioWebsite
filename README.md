# 🌐 Personal Portfolio Website

A clean, responsive, and modern portfolio website built using **HTML**, **CSS**, and **JavaScript**.  
This site showcases my skills, projects, resume, and contact information — designed with a simple UI and smooth animations.

---

## 🚀 Features

- Fully responsive layout  
- Clean and modern design  
- Smooth scrolling  
- Hover animations  
- Organized sections: Home, About, Projects, Contact  
- Easy to customize  
- Ready to deploy on GitHub Pages or Netlify  

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- *(Optional)* Live Server for local development
